Challenge to code objects validator with complex rules.

## Demo


## Install PNPM Package Manger

```sh
wget -qO- https://get.pnpm.io/install.sh | sh -
```

## Install All packages

```sh
pnpm install -r
```
## Run with Sample data in index.js

```sh
pnpm start
```

## Run Tests

```sh
pnpm test
```
